# Integrator for Moodle & NTU COOL

這是一個擴充功能，旨在幫助台科大與台大的跨校選課學生，將台科大 Moodle 的課程資訊直接整合進台大 COOL 的儀表板中。

## 功能特色
- **無縫整合**：在 NTU COOL 首頁直接顯示 NTUST Moodle 課程卡片。
- **自訂捷徑**：手動設定 Moodle 課程的公告、作業、討論區與教材連結。
- **視覺自訂**：可自由調整課程卡片的識別顏色與顯示/隱藏狀態。
- **隱私安全**：所有資料均儲存在本地瀏覽器，不會上傳至任何伺服器。

## 安裝說明
1. 下載本專案原始碼並解壓，確保`manifest.json` 在資料夾的一級目錄中。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入擴充功能」，並選擇包含 `manifest.json` 的資料夾。

## 使用方法
1. **獲取課程**：點選擴充功能設定中的按鈕前往 [NTUST Moodle 我的課程](https://moodle2.ntust.edu.tw/my/courses.php)，擴充功能會自動抓取您的課程列表。
2. **設定與自訂**：點擊瀏覽器工具列中的擴充功能圖示，開啟設定面板，填入相關連結並調整顏色。請務必填入完整 (帶有https/http前綴) 的連結
3. **查看結果**：重新整理 [NTU COOL 首頁](https://cool.ntu.edu.tw/)，即可看到整合後的成果。
